> 请求方式：post
>
> 接口描述：用于验证验证码，注册用户



#### 2.2.2 请求参数

参数格式：json

参数说明：

| 参数名 | 类型 | 是否必须 | 备注 |
| ---- | ---- | ---- | ---- |
| eamil | string | 必须 | 邮箱 |
| name | string | 必须 | 用户名 |

| password | string | 必须 | 密码 |

```json
{

}
```
